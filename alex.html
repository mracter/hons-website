Minimising Morphologies Needed to Evolve Co-operation

The research goal in my project was to find the minimal agent sensor configuration (morphology) that was able to evolve effective co-operation for the foraging task. This is a task where agents are required to collect square resources from an environment, and bring them back to the target area. We developed three variants of this task with each one requiring more cooperation to be effective; this was done by varying the ratios of three different box sizes, with each one requiring more agents pushing simultaneously to move it.

8 morphologies were tested in this project. The sensors used were chosen to be relatively realistic. Specifically, the sensors types tested were the ultrasonic sensor, and the typed proximity sensor:

  • Ultrasonic sensor: A simple high-range wide-angle proximity sensor that uses ultrasonic waves to detect the distance to the closest object.
  • Typed proximity sensors: A fairly low-range low-FOV sensor that can detect the distance and type of the nearest object.


Additionally, two more sensors were used in every morphology:

  • Bottom proximity sensor: a sensor on the bottom of the agent that can detect the specially-painted ground, and thereby whether it is in the target area or not.
  • Pickup sensor: a small sensor on the front of the agent that detects when a resource is in the grippers, and activates a heuristic to pick up the resource and bring it back to the target area.
