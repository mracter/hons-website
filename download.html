Simulator Source

The source code for the project is available on GitHub. Various different repositories were used:

  • hons-simulator: The simulator platform. The majority of the code written is available here.
  • hons-simulator/tree/GP: Alex's experiment platform and GP implementation.
  • hons-simulator/tree/neat_ex: Mary's homogeneous experiment platform.
  • hons-simulator/tree/hetero_comp: Mary's heterogeneous experiment platform.
  • hons-experiment: Jamie's experiment platform and NEAT-M implementation.

Additional Source Files

  • hons-website: The source for this website.
  • hons-docs: Some of the documentation produced over the course of this project.
